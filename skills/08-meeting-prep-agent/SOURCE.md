# Meeting Prep Agent

## Description
Generates a concise one-page briefing before any customer meeting — recent activity, open issues, stakeholder context, talking points, and risks to address.

## Audience
AEs, CSMs

## Input
Account name

## MCP Tools Used
- find_account
- get_account_status
- get_recent_account_activity
- get_engaged_people
- ask_sales_ai_about_account

## Project Knowledge Files
- None
