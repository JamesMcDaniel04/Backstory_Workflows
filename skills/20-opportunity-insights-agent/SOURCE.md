# Opportunity Insights Agent

## Description

Builds interactive pipeline intelligence dashboards that classify every open opportunity into Stalled, At Risk, Hidden Upside, or Acceleration insights — surfacing critical deal patterns days before they're obvious in CRM.

## Audience

- AEs
- CSMs
- Sales Leaders
- RevOps

## Input

"Build my pipeline dashboard" or account/region name

## MCP Tools Used

- top_records
- find_account
- get_opportunity_status
- get_recent_opportunity_activity
- get_account_status
- get_engaged_people
- ask_sales_ai_about_opportunity
