# Handoff Agent

## Description
Generates comprehensive account handoff documents when a rep or CSM transitions — relationship history, stakeholder map, open issues, deal context, and personality insights from communication tone.

## Audience
AEs, CSMs, Sales managers

## Input
Account name

## MCP Tools Used
- find_account
- get_account_status
- get_recent_account_activity
- get_engaged_people
- ask_sales_ai_about_account
- get_opportunity_status
- ask_sales_ai_about_opportunity

## Project Knowledge Files
- None
