# Executive Briefing Agent

## Description
Internal-facing executive summary for when leadership asks "what's going on with this account?" — pipeline risk, revenue impact, and strategic posture in one concise brief.

## Audience
Sales leaders, VPs, CROs

## Input
Account name

## MCP Tools Used
- find_account
- get_account_status
- get_recent_account_activity
- get_engaged_people
- ask_sales_ai_about_account
- get_opportunity_status

## Project Knowledge Files
- None
