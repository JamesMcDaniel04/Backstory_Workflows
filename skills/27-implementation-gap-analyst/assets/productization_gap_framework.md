# Productization Gap Framework

## Coverage Levels

- validated
- recipe-only
- generic-only
- missing

## Score Every Gap On

- rollout risk
- customer demand
- reuse potential
- dependency count

## High-Value Gaps

- unblock multiple workflows
- remove manual mapping steps
- eliminate schema drift risk
- replace customer-specific logic with canonical adapters

## Decision Rule

Prioritize the gaps that increase repeatability across customers, not just the loudest single request.
