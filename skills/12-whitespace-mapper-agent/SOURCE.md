# Whitespace Mapper Agent

## Description
Identifies expansion opportunities by analyzing engaged stakeholders across departments and business units — surfaces where engagement exists but revenue doesn't.

## Audience
AEs, CSMs, Sales leaders

## Input
Account name

## MCP Tools Used
- find_account
- get_engaged_people
- get_account_status
- get_recent_account_activity
- ask_sales_ai_about_account

## Project Knowledge Files
- None
