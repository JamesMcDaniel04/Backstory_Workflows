# Meeting Signal Normalization

## Normalize These Fields

- meeting title
- start / end time
- attendees
- organizer / owner
- transcript summary
- action items
- customer / account association
- source IDs

## Common Sources

- Google Calendar
- Microsoft 365 / Teams
- Gong
- Zoom
- Otter
- Fireflies
- Fathom

## Primary Risks

- weak transcript quality
- missing attendee identity
- ambiguous account matching
- duplicate calendar + note-taker records
