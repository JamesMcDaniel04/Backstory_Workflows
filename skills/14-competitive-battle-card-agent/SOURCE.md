# Competitive Battle Card Agent

## Description
Generates live competitive battle cards when competitors surface in deal activity — positioning, differentiators, talk tracks, and objection handling from deal context.

## Audience
AEs, Sales managers

## Input
Account name (+ optional competitor name)

## MCP Tools Used
- find_account
- get_opportunity_status
- get_recent_opportunity_activity
- ask_sales_ai_about_opportunity
- get_account_status
- ask_sales_ai_about_account

## Project Knowledge Files
- None
