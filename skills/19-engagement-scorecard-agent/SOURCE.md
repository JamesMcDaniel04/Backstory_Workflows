# Engagement Scorecard Agent

## Description
Weekly/monthly engagement metrics report — trending up or down, who's going dark, who's newly active. Pure data-driven engagement tracking.

## Audience
AEs, CSMs, Sales managers

## Input
Account name

## MCP Tools Used
- find_account
- get_engaged_people
- get_recent_account_activity
- get_account_status
- ask_sales_ai_about_account

## Project Knowledge Files
- None
