# Delivery Surface Patterns

## Preserve

- message intent
- severity
- owner/action lines
- structured sections

## Adapt

- Slack: compact markdown or blocks
- Teams: chat-safe copy or adaptive cards
- Email: subject line plus HTML/plain text body
- Webhook: JSON envelope for downstream systems

## Fallback Model

- preferred destination
- backup destination
- failure reason
- retry rules
- quiet-hour overrides
