# Opportunity Agent

## Description
Creates actionable opportunity health dashboards — deal health scoring, stakeholder engagement analysis, risk assessment, and strategic outreach recommendations.

## Audience
AEs, Sales leaders

## Input
Account name

## MCP Tools Used
- find_account
- get_opportunity_status
- get_recent_opportunity_activity
- ask_sales_ai_about_opportunity
- get_account_status

## Project Knowledge Files
- opp.md (opportunity agent architecture and health scoring framework)
